Taylor Swift Song of the Day API

This api is designed to give you a random song of the day from taylor's catalouge. Send a request to link/api to recieve the song of the day.

This API contains all Taylor Swift songs that were officially released on one of her albums and randomly returns a song each day. This API is completely free and doesn't require a key to use.

Example code: 

                async function getSong() {
                    try {
                        const res = await fetch('/sotd');
                        const data = await res.json();
                        console.log(data)
                        //your code goes here
                    } catch(err) {
                        console.log(`Error ${err}`);
                    }
                };
                
                getSong();

Response data will be an object that looks like: 

                {
                    name: 'Love Story', 
                    youtube: 'https://www.youtube.com/watch?v=8xg3vE8Ie_E', 
                    album: 'Fearless', 
                    taylorsversion: 'no',
                    lyrics: `We were both young when I first saw you
                    I close my eyes and the flashback starts
                    I'm standing there....`
                }

All songs include a youtube link from Taylor Swift's official youtube channel. Taylor's Versions of songs will be used when possible, except for songs with a music video to the original recording. Some of the videos on Taylor's channel are lyric videos. In those instances, the lyrics property will be an empty string.

If you would like to embed the youtube video in an iframe like the example below you will have to change the link to work with youtube's player api. Here's a javascript example of how to do this: 

                let embeded = data.youtube.replace('watch?v=', 'embed/')

Please note that the 'taylorsversion' property will be yes for any masters that she owns in the video sent. So, for albums like folklore and evermore that were always her oringal masters the taylorsversion property will be set to 'yes'.
